[[MCQ]]
question: Which method is most efficient for solving $x^2 - 5x + 6 = 0$?
A: Quadratic formula
B: Factoring ✓
C: Completing the square
D: Graphing
explanation: The two numbers that multiply to 6 and add to −5 are −2 and −3, so factoring is quickest here.

[[MCQ]]
question: What is the discriminant of $2x^2 + 3x - 2 = 0$?
A: 7
B: 17
C: 25 ✓
D: -7
explanation: Discriminant = b² - 4ac = 9 - 4(2)(-2) = 9 + 16 = 25.

[[FILL]]
question: Complete the square — $x^2 - 6x + \square = (x - 3)^2$
answer: 9
tolerance: 0
explanation: Half of −6 is −3, and (−3)² = 9.

[[FILL]]
question: Solve $x^2 - 9 = 0$. Give the positive root.
answer: 3
tolerance: 0
explanation: x² = 9, so x = ±3. The positive root is 3.

[[FILL]]
question: If $x^2 + bx + 16 = (x + 4)^2$, what is $b$?
answer: 8
tolerance: 0
explanation: (x + 4)² = x² + 8x + 16, so b = 8.

[[SHORT]]
question: Solve $2x^2 + 5x - 3 = 0$ using the quadratic formula. Show your full working.
marks: 3
answer: |
  Using the quadratic formula with a = 2, b = 5, c = −3:
  
  x = (−5 ± √(25 + 24)) / 4
    = (−5 ± √49) / 4
    = (−5 ± 7) / 4
  
  x = (−5 + 7)/4 = 2/4 = 1/2
  or
  x = (−5 − 7)/4 = −12/4 = −3
  
  ∴ x = 1/2 or x = −3

[[SHORT]]
question: The equation $x^2 + kx + 9 = 0$ has exactly one solution. Find all possible values of $k$.
marks: 3
answer: |
  For exactly one solution, the discriminant = 0:
  
  b² − 4ac = 0
  k² − 4(1)(9) = 0
  k² = 36
  k = ±6
  
  ∴ k = 6 or k = −6
